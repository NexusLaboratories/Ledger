## Managing Accounts

Accounts represent your financial sources (bank accounts, cash, credit cards, etc.).

### Creating an Account

1. Navigate to **Accounts** from the drawer
2. Tap the **+** (Add) button
3. Enter account details:
   - **Name**: e.g., "Chase Checking", "Cash Wallet"
   - **Currency**: Select from supported currencies (USD, EUR, GBP, etc.)
   - **Description**: Optional notes about the account
   - **Initial Balance**: Starting amount
4. Tap **Save**

### Viewing Account Details

- Tap any account to see:
  - Current balance
  - Transaction history
  - Quick actions (edit, delete)

### Editing or Deleting Accounts

1. Tap the account
2. Use the edit icon to modify details
3. Use the delete icon to remove (note: this will affect related transactions)

## Recording Transactions

Transactions are the heart of your financial tracking.

### Adding a Transaction

1. Navigate to **Transactions** or tap the **+** button on the Dashboard
2. Fill in the transaction details:
   - **Title**: Brief description (e.g., "Grocery Shopping")
   - **Amount**: Transaction value
   - **Type**: Income or Expense
   - **Account**: Which account this affects
   - **Category**: Organize by type (optional)
   - **Tags**: Add custom labels (optional)
   - **Date**: When the transaction occurred
   - **Notes**: Additional details (optional)
3. Tap **Save**

### Transaction Types

- **Expense**: Money going out (groceries, bills, entertainment)
- **Income**: Money coming in (salary, refunds, gifts)

### Viewing Transaction Details

1. Go to **Transactions**
2. Tap any transaction to view full details
3. Edit or delete as needed

### Filtering Transactions

Use the filter options to:
- View by date range
- Filter by account
- Filter by category
- Filter by transaction type
- Search by title

## Setting Budgets

Budgets help you stay on track with your spending goals.

### Creating a Budget

1. Navigate to **Budgets**
2. Tap the **+** button
3. Configure your budget:
   - **Name**: Budget title (e.g., "Monthly Groceries")
   - **Amount**: Budget limit
   - **Period**: Monthly, Quarterly, Yearly, or Custom
   - **Category**: Specific category or overall budget
   - **Start Date**: When the budget begins
   - **End Date**: For custom periods (optional)
4. Tap **Save**

### Budget Types

- **Overall Budget**: Track total spending across all categories
- **Category Budget**: Limit spending in specific categories (e.g., $500/month for dining)

### Monitoring Budget Progress

The app shows:
- **Amount Spent**: How much you've used
- **Remaining**: What's left
- **Percentage**: Visual progress indicator
- **Status**: On track, approaching limit, or exceeded

### Budget Notifications

Enable alerts when you reach:
- 50% of budget
- 80% of budget
- 90% of budget

Configure in **Settings → Notifications**

## Import & Export

Backup and restore your data with ease.

### Exporting Data

1. Navigate to **Settings → Data Management**
2. Tap **Export Data**
3. Choose what to include:
   - All accounts
   - All transactions
   - Categories and tags
   - Budgets
4. Data is exported as JSON format
5. Share via:
   - Email
   - Cloud storage (Google Drive, iCloud, Dropbox)
   - Save to device

### Import Data

1. Navigate to **Settings → Data Management**
2. Tap **Import Data**
3. Select the JSON file
4. Choose import mode:
   - **Merge**: Add imported data to existing data
   - **Replace**: Delete all data and import fresh (use with caution!)
5. Review import summary
6. Tap **Confirm**

### Export File Format

Exports include:
- Format version for compatibility
- Timestamp of export
- App version
- All your financial data in human-readable JSON

### Best Practices

- **Regular Backups**: Export your data monthly
- **Secure Storage**: Keep exports in secure cloud storage
- **Version Control**: Keep multiple backup versions
- **Before Major Changes**: Always export before large imports or app updates

## Notifications & Reminders

Stay informed about your finances with smart notifications.

### Report Reminders

Get periodic reminders to check your financial reports:

1. Navigate to **Settings → Notifications**
2. Enable **Report Reminders**
3. Set frequency:
   - Monthly (default)
   - Every 2 months
   - Quarterly (3 months)
   - Semi-annually (6 months)
   - Annually (12 months)
4. Notifications remind you to review spending trends

### Budget Notifications

Get alerted when approaching budget limits:

1. Navigate to **Settings → Notifications**
2. Enable **Budget Notifications**
3. Choose thresholds:
   - **50%**: Early warning
   - **80%**: Approaching limit
   - **90%**: Nearly exceeded
4. Receive notifications as you reach each threshold

### Notification Behavior

- Notifications appear in your device's notification center
- Tap a notification to open the relevant screen
- Notifications work even when the app is closed
- Requires notification permissions (granted on first use)

## Security & Privacy

Your financial data is precious. Ledger takes security seriously.

### Data Encryption

- **SQLCipher**: All data encrypted at rest using SQLCipher
- **Password Protected**: Database requires password to access
- **Secure Storage**: Passwords and preferences stored in device keychain

### Local-First Design

- **No Cloud Sync**: All data stays on your device
- **No Accounts Required**: No registration or login needed
- **Privacy by Default**: Your data never leaves your device (except manual exports)

### Biometric Authentication

- Use fingerprint or Face ID for quick, secure access
- Falls back to password if biometric fails
- Configure in **Settings → Security**

### Best Security Practices

1. **Set a Strong Password**: Use a unique, complex database password
2. **Enable Biometrics**: For convenient secure access
3. **Regular Backups**: Export data regularly to secure cloud storage
4. **Keep App Updated**: Install updates for security patches
5. **Don't Share Exports**: Backup files contain unencrypted data

## Tips & Best Practices

### Getting Accurate Financial Picture

1. **Record All Transactions**: Even small purchases add up
2. **Categorize Consistently**: Use the same categories for similar expenses
3. **Regular Updates**: Enter transactions daily or weekly
4. **Reconcile Accounts**: Compare app balances with actual accounts monthly

### Effective Budgeting

1. **Start Conservative**: Set achievable budget limits
2. **Track and Adjust**: Review budgets monthly and adjust as needed
3. **Use Categories**: Create category-specific budgets for better control
4. **Enable Alerts**: Turn on budget notifications to stay aware

### Organization Tips

1. **Use Tags Strategically**: Don't over-tag; keep it simple
2. **Create Subcategories**: For detailed tracking without clutter
3. **Descriptive Titles**: Make transaction titles clear and searchable
4. **Add Notes**: Include details you might need later (receipt numbers, etc.)

### Workflow Suggestions

**Daily**:
- Record new transactions as they happen
- Check recent transactions for accuracy

**Weekly**:
- Review spending for the week
- Verify all transactions are categorized
- Check budget progress

**Monthly**:
- Generate and review monthly report
- Reconcile accounts with bank statements
- Export backup
- Adjust budgets for next month
- Review and clean up categories/tags

**Quarterly**:
- Analyze spending trends
- Plan budget changes
- Review financial goals

### Troubleshooting Common Issues

**Can't see recent transactions**:
- Check filter settings
- Verify correct account is selected
- Try refreshing the screen

**Budget not tracking correctly**:
- Ensure transactions are categorized properly
- Check budget date range
- Verify expense type (not income)

**Reports showing unexpected data**:
- Review date range selection
- Check account filters
- Ensure all transactions are properly dated

**Import failed**:
- Verify JSON file format is correct
- Check file is not corrupted
- Ensure file was exported from compatible version
